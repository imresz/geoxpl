# geoxpl

Present a web page with a search bar. The initial map should Victorian unless some location information is available 
A set of results is presented from which the use either selects one or chooses the option for further searching.
Choosing the further search option triggers more background pre-processing and or more raw data sourcing. Either a result is returned or a message to try again in a little while.
Choosing one of the returned options displays the feature on an interactive map. 
All the normal map functionality should be enabled.

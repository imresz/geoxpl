# geoxpl

River

Represent a river as a directed graph. Use graph traversal to identify the main stem and calculate the longest upstream path.

Road

Represent a road as single or double solid line.
Include a route number if known.
Show all roads from a route number.


Mountain
Represent a mountain as a dot for the peak and the height.
Attempt to identify a base or an extent.
Draw the extend as a surrounding polygon.

Mountain Range
Identify the extent and a confidence level for the extent.
Attempt to identify any sub ranges.
Attempt to identify a parent range.
Identify and draw a ridge line.
Represent a range as a polygon and a ridge line. Use a solid line if the extent is certain or a dotted line if the extent is uncertain.

Valley
The extent of a valley will understood to be approximate.
Identify or infer the extent and provide a confidence level.
Represent a valley as a polygon and show a confidence level.
Show any large rivers associated the vallye within the valley extent.
Show any mountain ranges surrounding the valley extent.
